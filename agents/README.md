# CLI Agents

This project includes CLI-style agent definitions in this folder.

## Included agents

- `codex-cli.json`
- `claude-code-cli.json`
- `copilot-cli.json`

## Notes

- These are lightweight launch/config definitions intended for orchestration.
- Update `command` and `args` if your local CLI install uses different names.
- `registry.json` provides a single list of all agent config files.
